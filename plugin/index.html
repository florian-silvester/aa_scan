⚙️ Step 1: Configure Credentials

🔐 Set up your credentials via environment variables:
  • 📝 SANITY_PROJECT_ID: Your Sanity project ID
  • 📝 SANITY_DATASET: Your Sanity dataset (usually 'production')
  • 📝 SANITY_API_TOKEN: Your Sanity API token
  • 📝 OPENAI_API_KEY: Your OpenAI API key
  • 🟡 DROPBOX_TOKEN: Optional Dropbox access token

Create a .env file in the project root with these variables, or set them in your environment.

See .env.example for the template.
